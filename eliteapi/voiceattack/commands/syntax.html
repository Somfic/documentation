How does EliteVA work with in-game events?

Almost every action (or event) that's able to be performed in Elite: Dangerous gets logged by the game in Journal Files. EliteVA constantly scans these files while you're playing the game and outputs all this information to VoiceAttack.

In this section we will focus on how EliteVA handles in-game events, and how you're able to hook into in-game events through VoiceAttack's voice command system.

Subscribing to an in-game event

Whenever you perform an action in Elite: Dangerous an event is triggered by the EliteVA plugin. Hooking into these events can be done through "fake" voice commands.

EliteVA will detect voice commands with a specific syntax, and automatically link them to the correct in-game event.
